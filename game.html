Game "CATVENTURE"

About the project
For my programming assignment, I created a game called CatVenture. It’s a breakout-style game that you control using keyboard buttons, but with a fun twist—it's all about cats! Since I love cats, I wanted to bring that theme into my project, making it both playful and visually appealing. Designing the game mechanics and adding cat-themed elements made the process really exciting. From the cozy atmosphere to the engaging gameplay, CatVenture is a game that combines my love for cats with my interest in game development, making it a truly fun and personal project.
♡ Anna Volosheniuk 2025
